# 💰 Money Tracker Challenge

Chào mừng bạn đến với thử thách phát triển ứng dụng **Money Tracker**! Repo này được thiết kế để giúp bạn (hoặc team của bạn) bắt đầu code ngay lập tức với một Product Backlog đã được chuẩn bị sẵn theo tiêu chuẩn Agile.

## 🎯 Mục tiêu
Hoàn thành bộ tính năng quản lý tài chính cá nhân từ cơ bản đến nâng cao. Bạn có thể tự chọn Tech Stack (React, Vue, Flutter, Spring Boot, Node.js, v.v.) tùy theo sở thích và mục tiêu học tập.

## 🚀 Cách bắt đầu
1. **Fork/Clone repo này**: Để bắt đầu quản lý trên tài khoản của bạn.
2. **Kiểm tra Tab Issues**: Toàn bộ 50 User Stories đã được tạo sẵn dưới dạng GitHub Issues.
3. **Thiết lập GitHub Project**: 
   - Vào tab `Projects` -> `New Project`.
   - Chọn template `Board` hoặc `Table`.
   - Add toàn bộ Issues từ repo này vào Project để quản lý Sprint.
4. **Code & Pull Request**: Thực hiện theo quy trình chuyên nghiệp: Tạo nhánh -> Code -> PR -> Review.

## 📋 Cấu trúc Repo
- `/tasks`: Chứa file `README.md` mô tả chi tiết Product Backlog.
- `.github`: Chứa các template cho Issue và Pull Request.
- `CONTRIBUTING.md`: Hướng dẫn dành cho team 3-5 người làm việc chung.

## 🛠️ Tech Stack Gợi ý
- **Frontend**: React (TailwindCSS, Shadcn/UI).
- **Backend**: Node.js (Express/NestJS) hoặc Spring Boot.
- **Database**: PostgreSQL hoặc MongoDB.
